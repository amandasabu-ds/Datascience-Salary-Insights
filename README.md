# Datascience-Salary-Insights

# Overview
This project is an introductory exploratory data analysis (EDA) conducted as part of early academic training in Data Science and Analytics. The analysis focuses on understanding salary trends and employment patterns in data science roles.

# Dataset
The dataset contains information about:
- Job roles in data science
- Salary distributions
- Employment types
- Experience levels

# Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

# Analysis Performed
- Data loading and preprocessing
- Analysis of salary trends
- Comparison of salaries across job roles and employment types
- Basic visualizations to identify patterns and insights

# Key Learnings
- Hands-on experience with real-world datasets
- Improved understanding of data preprocessing and visualization
- Practical application of Python libraries for data analysis
